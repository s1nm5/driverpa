# Driverpa - Простая программа для установки драйверов в виде EXE файла.

## Необходимые средства для работы:
- WinRAR или 7-zip (возможен вариант и без, если уже всё подготовлено);
- Драйвера для принтеров, в виде архива, либо в виде EXE файла.

## Установка дополнтельных драйверов:
1. Открывем EXE нашей программы через WinRAR;
2. Закидываем EXE драйвер в папку "Driver";
3. Выносим файл "menu main.bat" на рабочий стол;
4. Открываем файл "menu main.bat" через блокнот;
5. Редактируем код под свой драйвер, в "::" написан комментарий к коду, кто за что отвечает (делаем по примеру);
6. Закидываем файл "menu main.bat" обратно в архив;
7. Сохраняем, пользуемся.

## Как сделать архив с драйвером в EXE файл:
1. Разархивируем драйвера;
2. Выделяем все файлы в папке;
3. Нажимем правой кнопкой мыши и выбираем пункт "Добавить в архив", через WinRAR;
4. Выбираем в "Общие" пункт "Создать самораспаковывающийся архив";
5. Заходим в "Дополнительно" пункт "Параметры SFX":
	- "Общие" пункт "Путь распаковки" - %TEMP%\папка, пункт "Абсолютный путь";
	- "Установка" пункт "Выполнить после распаковки" - названия файла с его расширением;
	- Пункт "Тихий режим" - "Скрыть всё".
6. Сохраняем, закрываем.

## Ошибки открытия:
- Открываем через WinRAR;
- Переносим все данные в обычную папку;
- Открываем файл "menu main.bat";
- Пользуемся.
